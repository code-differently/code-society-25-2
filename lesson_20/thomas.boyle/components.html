Introduction

This page showcases three mobile‑responsive UI controls built with only HTML, CSS, and vanilla JavaScript. It runs fully offline—no external fonts, libraries, or image files.

Tabbed Component

Keyboard: Left / Right, Home / End

Tabs let users switch between related content sections without leaving the page. This version uses ARIA roles and keyboard interactions.

  • No external dependencies
  • Progressive enhancement
  • Roving tabindex pattern

Arrow keys move focus across tabs; aria-selected updates announce the active tab to assistive technologies.

Accordion Component

Keyboard: Up / Down to move, Enter / Space to toggle

Accordions reveal or hide related content sections, conserving vertical space.

Use arrow keys to move between headers; press Enter or Space to toggle a section.

Modify the script to allow only one open section at a time if desired.

Rotating Photo Gallery

Shows 1–3 images depending on width. Auto-rotates every 4s; pauses on hover/focus.

Offline Operation

All images are generated on the fly as inline SVG data URIs. No network requests are required.
