Tabbed Component

Keyboard: Left / Right, Home / End

Tabs let users switch between related content without leaving the page.

  • No external dependencies
  • Progressive enhancement
  • Roving tabindex pattern

Keyboard and ARIA attributes ensure accessibility support.

Accordion Component

Keyboard: Up / Down to move, Enter / Space to toggle

Accordions reveal or hide related content sections.

Up/Down arrow keys move between triggers. Enter / Space toggles.

Can enforce single-open behavior by adjusting the script.

Rotating Photo Gallery

Shows 1–3 images (responsive). Auto-rotates every 4s; pauses on hover/focus.

Offline Operation

All images are generated on the fly as inline SVG data URIs. No network requests required.
