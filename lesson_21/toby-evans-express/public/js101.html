JavaScript 101 — Highlights

Key ideas: history and runtimes, variables and scope, objects/classes, async/event loop, promises and async/await, modules and npm, DOM APIs.

History and Runtimes

  • Created at Netscape; standardized as ECMAScript.
  • Runs in browsers and on servers via Node.js.

Variables and Scope

  • let/const are block scoped; avoid var.
  • Dynamic typing; closures capture outer scope.

Objects and Classes

  • Prototype-based inheritance; classes are syntactic sugar.
  • Built-ins: Array, Set, Map; weak collections help memory.

Async Model

  • Single-threaded event loop with callbacks, promises, async/await.
  • try/catch for async error handling.

Modules and npm

  • Export/import to organize code.
  • npm manages dependencies in package.json.

DOM and UI

  • document.querySelector, events with addEventListener.
  • Frameworks provide declarative, component-based UIs.
